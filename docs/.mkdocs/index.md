# Bitbox Developer Documentation

Welcome to the developer docs. This is the companion to the user guides, aimed at contributors who want to understand and extend the Bitbox API.

## What's inside

- A quick orientation so you know how the site is structured.
- A short guide on running and hot-reloading the docs locally.
- API reference pages that stay in sync with the code under `src/bitbox` via live docstrings.

When you add new modules or functions, document them with clear docstrings and add them to the navigation; the reference will pick them up automatically.
